# Pterodactyl Open.MP Egg

Custom Pterodactyl egg for open.mp servers.

## Features

- Automatic open.mp installation
- Direct binary startup
- Compatible with Pterodactyl Panel 1.x
- Linux x86 support

## Import

Admin Panel → Nests → Import Egg → upload egg-openmp.json

## open.mp Version

v1.5.8.3079
